# Airbnb Financial Analysis

## Project Summary
This project analyzes Airbnb stock performance using financial metrics such as return, volatility, beta, and market correlation. The analysis evaluates Airbnb as a high-growth but high-risk investment.

## Research Questions
- How has Airbnb performed over time?
- What is its volatility compared with the broader market?
- What does its beta reveal about market sensitivity?
- Is Airbnb suitable for risk-tolerant investors?

## Dataset Overview
The project uses historical stock data from Yahoo Finance and benchmark market data to measure return, volatility, and beta.

## Tools and Methods
- Yahoo Finance data
- Excel
- Power BI
- Return analysis
- Volatility analysis
- Beta calculation
- Market correlation analysis

## Key Findings
- Airbnb delivered an annualized return of about 11%.
- Average daily return was 0.000412.
- Airbnb’s annualized volatility was 45%, compared with 20% for the market.
- Airbnb’s beta was 1.58, showing it is 58% more volatile than the market.
- The stock had strong sensitivity to overall market movements.
- The projected trading range was 120 to 160.
- The stock is better suited for aggressive investors than conservative ones.

## Insights
- Airbnb has growth potential, but the risk level is high.
- High volatility makes the stock more suitable for short-term or tactical investing.
- Performance is closely linked to tourism recovery, earnings, and macroeconomic conditions.
- Diversification is important when holding a stock like Airbnb.
